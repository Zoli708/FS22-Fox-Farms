# 🦊 FS22 Fox Farms 🦊
Basic size US map with large fields.

What the map provides:
- 22 field, 4 meadows (0.2-33.6ha)
- 1 Farm
- Fuel stations
- Water filling stations
- Filling station (seed, fertilizer, etc.)
- Cow pasture (objects: LazyEM)
- Open bunker silos
- 8 sale points
- Animal dealer
- Vehicle shop
- BGA
- Mission system
